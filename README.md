# Exercícios de Front-End (React)

Repositório dedicado ao aprendizado contínuo de desenvolvimento Front-End com foco em **React**, TypeScript e ferramentas modernas do ecossistema.

---

## Objetivo

Desenvolver aplicações front-end modernas, com ênfase em:

* Componentização com React
* Gerenciamento de estado
* Consumo de APIs REST
* Tipagem com TypeScript
* Boas práticas de organização e escalabilidade

---

## Tecnologias Utilizadas

* React
* TypeScript
* JavaScript (ES6+)
* HTML5 e CSS3
* Vite ou Create React App
* Git e GitHub

## Como utilizar

1. Clone o repositório:

```bash id="t9v4az"
git clone <url-do-repositorio>
```

2. Instale as dependências:

```bash id="k2n7qw"
npm install
```

3. Execute o projeto:

```bash id="p5d8lx"
npm run dev
```

* Separação de responsabilidades

* Uso consistente de hooks
* Organização por domínio (feature-based)
